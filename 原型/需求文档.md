## 一、总体架构

```
┌─────────────────────────────┐
│       代办事项软件 (Windows)       │
│ ┌─────────────┐  ┌───────────┐ │
│ │ 任务管理模块 │  │ 提醒/通知模块 │ │
│ └─────────────┘  └───────────┘ │
│ ┌─────────────┐  ┌───────────┐ │
│ │ 可视化界面 │  │ 同步/导出模块 │ │
│ └─────────────┘  └───────────┘ │
└─────────────────────────────┘
            │
            ▼
  ┌───────────────────┐
  │  内部存储文件 (JSON/YAML) │  ← 软件唯一权威来源
  └───────────────────┘
            │
            ▼ （每天固定时间）
  ┌───────────────────┐
  │  Obsidian 文件夹  │  ← 生成 `YYYY-MM-DD.md`，Markdown 格式
  └───────────────────┘
```

## 二、核心功能模块

### 1. 任务管理模块

* **任务字段**：

  * 标题（必填）
  * 标签（可选）
  * 优先级（低/中/高）
  * 重复周期（每天/每周/每月）
  * 时间（HH:mm）
  * 状态（待办/完成/延迟）
* **任务录入方式**：

  * 托盘图标 → 弹窗输入 → 回车
  * 支持全局快捷键弹出输入框
* **任务操作**：

  * 编辑任务字段
  * 标记完成 / 延迟
  * 删除任务

### 2. 提醒/通知模块

* **方式**：Windows 右下角弹窗
* **弹窗操作**：

  * 点击标记完成
  * 点击延迟（可选择延迟时间，如 5/10/30 分钟）
* **触发逻辑**：

  * 根据任务时间精准触发
  * 支持重复任务
* **后台运行**：

  * 软件在系统托盘常驻
  * 提醒独立于 Obsidian 运行

### 3. 可视化界面模块

* **列表视图**：

  * 按天展示任务
  * 显示标题、标签、优先级、状态
  * 支持按标签/优先级筛选
* **日历视图**：

  * 周视图 / 月视图
  * 点击任务可操作：修改时间、完成状态、标签/优先级
* **界面设计**：

  * 简洁直观
  * 托盘入口 + 独立窗口切换

### 4. 同步/导出模块

* **生成文件**：

  * 每天生成 `YYYY-MM-DD.md`
  * Markdown 格式：

    ```markdown
    - [ ] 任务标题 #标签 @优先级
      时间: HH:mm
      重复: 每天/每周/每月
    ```
* **同步触发**：

  * 每天固定时间自动同步
* **文件组织**：

  * 放入 Obsidian 指定文件夹

## 三、数据存储

* **内部权威数据文件**：

  * 使用 JSON 或 YAML 存储所有任务
  * 示例 JSON：

    ```json
    [
      {
        "title": "提交周报",
        "time": "16:30",
        "tags": ["工作"],
        "priority": "高",
        "repeat": "每天",
        "status": "待办"
      }
    ]
    ```
* **优点**：

  * 轻量、易读
  * 便于扩展未来多端同步
  * 可手动备份

## 四、技术选型（推荐）

* **前端界面**：

  * Electron.js（桌面应用，跨 Windows / macOS 可扩展）
  * 或者 PyQt / PySide（Python，快速开发）
* **系统通知**：

  * Electron：`electron-notifications` 或 `node-notifier`
  * PyQt：`QSystemTrayIcon.showMessage`
* **数据存储**：

  * JSON 文件（轻量，可直接读写）
* **定时任务**：

  * 内置 Timer/Background Worker，每分钟扫描任务触发提醒
* **同步到 Obsidian**：

  * 生成 Markdown 文件，直接写入指定文件夹

## 六、开发优先级建议

1. **核心功能先行**：

   * 任务录入 → 系统级提醒 → 完成/延迟操作 → 内部文件存储
2. **可视化界面**：

   * 列表视图 → 日历视图 → 交互操作
3. **Obsidian 同步**：

   * Markdown 文件生成 → 每日定时同步
4. **优化和扩展**：

   * 快捷键、重复任务逻辑、界面美化、未来多端
